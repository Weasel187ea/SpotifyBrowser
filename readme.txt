Spotify Browser is a Web Application that presents the user with personalized music
preferences from Spotify. Interacting with the application allows for more details 
as well as discovery for musical artists, albums, and tracks. 
Note: Co created with classmate. In order to run, fill out client_secret with respective user spotify keys, found here (https://developer.spotify.com/dashboard/). 
Once filled, you can npm install both the client and webserver directories, then npm run both. 
This project is a copy of the repository due to private restrictions.
